Mewoo Database Schema

Complete Relational Architecture & Annotations for Presentation
users
PKidUUID
email, passwordVARCHAR
roleENUM
GISTlocationGEOGRAPHY(Point)
💡 Speaker Note: This is the central Auth Hub. We use PostGIS (GEOGRAPHY) here to store the user's home location so we can instantly calculate "Nearby Vets" without lagging the server.
vet_profiles
PKFKuser_idUUID
clinic_nameVARCHAR
is_emergencyBOOLEAN
statusENUM
💡 Speaker Note: By separating Vet & Trainer profiles from the main Users table, we keep the database normalized. If a vet deletes their account, the cascade deletes their profile instantly.
trainer_profiles
PKFKuser_idUUID
specialtiesTEXT[]
bioTEXT
💡 Speaker Note: Stores trainer-specific data. Specialties are stored as an array so owners can easily filter for "Obedience" or "Agility" training in the app.
pets
PKidUUID
FKowner_idUUID
name, speciesVARCHAR
avatar_urlVARCHAR
💡 Speaker Note: The core entity of the app. It links directly to the owner. Images are securely stored in external buckets (like S3 or Cloudflare) and only the URL is kept here to keep the database extremely fast.
appointments
PKidUUID
FKpet_idUUID
FKvet_user_idUUID
ai_triage_summaryTEXT
💡 Speaker Note: This bridges a Pet and a Vet. Notice the "AI Triage Summary" column! This is where our Agentic AI saves its analysis of the pet's symptoms before the vet even sees them.
lost_pets
PKidUUID
FKpet_idUUID
GISTlost_locationGEOGRAPHY
statusENUM
💡 Speaker Note: The spatial engine shines here. By GIST-indexing the exact GPS coordinates where the pet was lost, we can instantly trigger push notifications to all users currently standing within a 5-mile radius.
services
PKidUUID
FKprovider_idUUID
base_priceDECIMAL
categoryENUM
💡 Speaker Note: Powers the marketplace. Trainers and Walkers define their offerings here. Using precise DECIMAL types ensures financial accuracy.
payments
PKidUUID
FKbooking_idUUID
gateway_transactionVARCHAR
amountDECIMAL
💡 Speaker Note: Designed for regional gateways (like Moyasar or Paymob). We store the secure transaction ID provided by the bank, linking the Pet Owner's payment directly to the Trainer's completed booking.
